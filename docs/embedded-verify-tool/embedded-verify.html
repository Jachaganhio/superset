Superset Embedded Verify

Step-by-step verification of the embedded dashboard flow

Help & Troubleshooting

Purpose

This tool verifies that your Apache Superset instance is correctly configured for embedded dashboards. It walks through authentication, guest token generation, and dashboard embedding using @superset-ui/embedded-sdk.

Prerequisites

  • Superset with EMBEDDED_SUPERSET feature flag enabled (default: True)
  • A dashboard with embedding configured: set Allowed Domains in the Embed Dashboard modal
  • A user with can_grant_guest_token permission (admin role by default)
  • CORS configured on Superset to allow requests from this tool's origin

Finding Your Dashboard ID

Open a dashboard in Superset. The URL contains the dashboard ID: /superset/dashboard/1/ — 1 is the ID. You can also use the API: GET /api/v1/dashboard/

CORS Configuration

Add the following to superset_config.py:

ENABLE_CORS = True
CORS_OPTIONS = {
  "origins": [
    "http://localhost:8080",
    "https://your-tool-domain.com",
  ],
  "supports_credentials": True,
}

Mixed Content

If your Superset instance uses HTTPS, this tool must also be served over HTTPS. Browsers block mixed content (HTTPS page loading HTTP iframes).

Guest Token Expiration

The default guest token TTL is 5 minutes (GUEST_TOKEN_JWT_EXP_SECONDS). The SDK automatically refreshes the token. If verification fails after initial success, the token may have expired — restart the verification.

Checking Embed Configuration

To verify a dashboard is configured for embedding via the API:

curl -H "Authorization: Bearer <token>" \
  http://localhost:8088/api/v1/dashboard/1/embedded

The response includes uuid (the embedded dashboard UUID), dashboard_id, and allowed_domains.

Troubleshooting Steps

  • 401 Unauthorized — Check your username and password
  • 403 Forbidden — User lacks can_grant_guest_token permission, or dashboard is not accessible
  • 404 Not Found — Dashboard ID does not exist or embedding is not configured
  • Failed to fetch — Likely a CORS issue. Configure CORS on Superset or serve the tool from the same origin
  • Iframe shows "Not Found" — The embedded UUID is incorrect. Check the Embed Dashboard modal
